# 聊天人格生成器

## 概述

人格生成器通过 4 阶段 AI 流水线，为 Bot 创建详细的、结构化的聊天人格提示词。生成的人格定义了 Bot 的个性、沟通风格和身份信息，保存后可分配给任意 Bot。

## 生成流水线

| 阶段 | 内容 |
|------|------|
| 1. 语气与人格 | 根据关键词提炼核心性格特征 |
| 2. 角色画像 | 详细的人物性格描述 |
| 3. 聊天提示词 | 对话行为规则 |
| 4. 具体聊天风格 | 融合身份属性的最终模板填充 |

## 身份属性

生成前可选填以下属性：

| 属性 | 可选项 |
|------|--------|
| 星座 | 白羊座 → 双鱼座（12 个） |
| 属相 | 鼠 → 猪（12 个） |
| 性别 | 男性 / 女性 / 其他 |
| 年龄感 | 少年感 / 年轻成年人 / 成熟 / 中年感 |

这些属性会作为补充说明追加到生成关键词中，并**直接嵌入到保存的 Prompt 内容首行**：

```
[身份设定] 星座：双鱼座、属相：羊、性别：女性、年龄感：少年感。
如果用户问你的星座、属相、性别、年龄等个人特征，直接如实回答。
```

注入聊天系统 Prompt 时，此头部会展开为明确的陈述句（"你的星座是双鱼座，..."），并强制禁止否认身份。

## 使用方法

1. 技能 → 聊天 → 人格生成器
2. 填写描述性关键词（如"活泼、喜欢星星、话比较多"）
3. 可选：选择星座 / 属相 / 性别 / 年龄感
4. 点击 **生成**
5. 查看结构化卡片预览（或切换到原始文本编辑模式）
6. 填写名称，点击 **保存**
7. 在下方 Prompt 管理区选择 Bot 并点击 **分配**

## 保存与分配

- 保存的 Prompt 存入 `user_prompts` 表（`type = 'chat'`）
- Prompt 内容 = `[身份设定]` 头部 + 生成的正文
- 每个 Bot 可绑定一个聊天 Prompt（`bot` 表的 `chat_prompt_id` 字段）
- 重新分配或删除 Prompt 会自动同步到 Bot 列表
